# User Management System Gin Gonic
Simple REST Api yang dibuat menggunakan framework gin gonic.

## Features
1. Create User: Register new users with unique emails.
2. Read All Users: List all users currently in the system.
3. Read User by ID: Fetch specific user details.
4. Update User: Partial updates (PATCH) for email or password.
5. Delete User: Remove users from the system by ID.
